ubuntu18.04-install-openjdk8

Posted on 2019-05-25 |

添加openjdk8的第三方源

sudo add-apt-repository ppa:openjdk-r/ppa
ppa (Personal Package Achives)

执行更新

apt-get update

安装openjdk8

sudo apt-get install openjdk-8-jdk

选择版本

sudo update-alternatives - -config java

确认安装成功

java -version
